# Agent Skills

这是一个个人使用的技能库，存储了供 ClaudeCode/OpenCode/Codex/Gemini CLI/Antigravity 使用的自定义技能（Skills）。每个技能都包含一套专门的指令，用于扩展 AI 代理在特定领域（如项目分析、文档编写、自动化流程等）的能力。

## 技能索引

| 技能名称                     | 核心用途简述                                         | 目录路径                                                               |
| :--------------------------- | :--------------------------------------------------- | :--------------------------------------------------------------------- |
| `claude-md-expert`           | 创建、更新和优化 CLAUDE.md 配置文件。                | [skills/claude-md-expert](skills/claude-md-expert)                     |
| `crafting-effective-readmes` | 根据受众和项目类型编写或改进 README 文件。           | [skills/crafting-effective-readmes](skills/crafting-effective-readmes) |
| `daily-progress`             | 从 git 记录自动生成每日/每周工作进展报告。           | [skills/daily-progress](skills/daily-progress)                         |
| `defuddle`                   | 使用 Defuddle CLI 从网页中提取纯净的 Markdown 内容。 | [skills/defuddle](skills/defuddle)                                     |
| `diagram-expert`             | 使用 Mermaid、PlantUML 等工具创建各类技术图表。      | [skills/diagram-expert](skills/diagram-expert)                         |
| `find-skills`                | 帮助发现并安装生态系统中的其他代理技能。             | [skills/find-skills](skills/find-skills)                               |
| `humanizer-zh`               | 去除文本中的 AI 生成痕迹，使文字听起来更自然。       | [skills/humanizer-zh](skills/humanizer-zh)                             |
| `repo-explorer`              | 深度探索项目架构、目录结构、核心模块和依赖关系。     | [skills/repo-explorer](skills/repo-explorer)                           |
| `web-security-expert`        | 基于 OWASP Top 10 2025 提供 Web 安全审计和修复建议。 | [skills/web-security-expert](skills/web-security-expert)               |
