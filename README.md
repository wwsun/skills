# Gemini CLI 技能库 (Agent Skills)

这是一个个人使用的技能库，存储了供 Gemini CLI 使用的自定义技能（Skills）。每个技能都包含一套专门的指令，用于扩展 AI 代理在特定领域（如项目分析、文档编写、Obsidian 增强等）的能力。

## 技能索引

| 技能名称 | 核心用途简述 | 目录路径 |
| :--- | :--- | :--- |
| `repo-explorer` | 深度探索项目架构、目录结构、核心模块和依赖关系。 | [skills/repo-explorer](skills/repo-explorer) |
| `ai-sdk` | 关于 Vercel AI SDK 的问答及功能构建指南。 | [skills/ai-sdk](skills/ai-sdk) |
| `crafting-effective-readmes` | 根据受众和项目类型编写或改进 README 文件。 | [skills/crafting-effective-readmes](skills/crafting-effective-readmes) |
| `defuddle` | 使用 Defuddle CLI 从网页中提取纯净的 Markdown 内容。 | [skills/defuddle](skills/defuddle) |
| `find-skills` | 帮助发现并安装生态系统中的其他代理技能。 | [skills/find-skills](skills/find-skills) |
| `humanizer-zh` | 去除文本中的 AI 生成痕迹，使文字听起来更自然。 | [skills/humanizer-zh](skills/humanizer-zh) |
