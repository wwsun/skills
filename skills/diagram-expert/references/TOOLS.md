# Diagramming Tools Reference

## Complete Tool List

### Popular Tools
- **Mermaid** - Versatile, GitHub/markdown-friendly
- **PlantUML** - Comprehensive UML support
- **GraphViz** - Graph layouts and network diagrams
- **Ditaa** - ASCII art to diagrams
- **D2** - Modern declarative language

### Specialized Tools
- **BPMN** - Business process modeling
- **C4** - Software architecture
- **DBML** - Database schemas
- **Erd** - Entity-relationship diagrams
- **Excalidraw** - Hand-drawn style
- **WaveDrom** - Digital timing diagrams
- **Vega/Vega-Lite** - Data visualization

### Other Options
BlockDiag, Bytefield, Nomnoml, Pikchr, Structurizr, SvgBob, Symbolator, TikZ, UMLet, WireViz

## Diagram Type Mapping

### Common Graphs
- **Block Diagram** → BlockDiag, Ditaa, Mermaid
- **DAG (Directed Acyclic Graph)** → GraphViz, Mermaid
- **Mind Map** → PlantUML, Mermaid
- **Flowchart** → Mermaid, PlantUML

### UML/C4 Diagrams
- **Sequence Diagram** → Mermaid, PlantUML
- **E-R Diagram** → Erd, Mermaid, DBML
- **Activity Diagram** → Mermaid, PlantUML
- **Use Case Diagram** → PlantUML
- **C4 Model** → PlantUML, Structurizr
- **Class Diagram** → Mermaid, PlantUML

### Project Management
- **Gantt Chart** → Mermaid, PlantUML
- **Business Process** → BPMN
- **WBS (Work Breakdown Structure)** → PlantUML

### Technical Diagrams
- **Network Topology** → GraphViz, D2
- **Database Schema** → DBML, Erd
- **Digital Timing** → WaveDrom
- **Wiring/Circuit** → WireViz
- **Data Visualization** → Vega, Vega-Lite

## Tool Selection Criteria

**Choose Mermaid when:**
- Need markdown/GitHub compatibility
- Creating common diagram types
- Want simple syntax
- Need quick results

**Choose PlantUML when:**
- Need comprehensive UML features
- Creating complex class diagrams
- Need C4 architecture diagrams
- Require extensive customization

**Choose GraphViz when:**
- Creating network topologies
- Need precise graph layouts
- Working with large interconnected systems
- Need DOT language features

**Choose specialized tools when:**
- BPMN: Business process modeling required
- WaveDrom: Digital timing diagrams needed
- DBML: Database schema as code
- Ditaa: ASCII art aesthetic desired
